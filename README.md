# Oasis Python Internship Projects

This repository contains the Python projects completed as part of my **Python Programming Internship at Oasis Infobyte**.

## Projects

### 📌 Project 2: BMI Calculator

A command-line Python application that calculates the Body Mass Index (BMI) based on the user's weight and height and classifies the result into standard BMI categories.

**Features**

* User-friendly input
* Accurate BMI calculation
* BMI category classification
* Simple command-line interface

**Technologies Used**

* Python

---

### 📌 Project 3: Random Password Generator

A command-line Python application that generates secure random passwords according to the user's preferred password length.

**Features**

* User-defined password length
* Random password generation
* Uses letters, numbers, and special characters

**Technologies Used**

* Python
* `random`
* `string`

---

### 📌 Project 4: Weather App

A command-line weather application that retrieves real-time weather information for a user-specified city using a weather API.

**Features**

* Search weather by city
* Displays temperature
* Displays humidity
* Displays weather conditions
* API integration

**Technologies Used**

* Python
* Requests library
* Weather API
